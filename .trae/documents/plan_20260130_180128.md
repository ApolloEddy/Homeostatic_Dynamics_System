## 关键要求执行

### 参数保护原则
- 所有代码中的实际参数（如0.55, 0.28, 0.04等）均用伪变量替代
- 例如：`baseline_da = μ_da`，并在公式下方解释
- 配置文件中的数值用符号表示

### 多学科融合
- **神经学**：多巴胺、去甲肾上腺素、血清素的生物学功能
- **心理学**：内驱力理论、情绪调节、人格稳定性
- **社会行为学**：社交互动、边界设定、关系维护
- **数学**：随机微分方程、稳定性分析、参数优化
- **计算机科学**：系统架构、API设计、数据结构

### 文档互跳设计
- README → 目录 → 各章节
- 章节间上一篇/下一篇导航
- 公式与代码引用链接

### 语气风格
- 客观严谨的学术风格
- 使用专业术语但提供解释
- 避免主观评价和炫技叙述
