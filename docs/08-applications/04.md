# 08.4 可解释性分析
### 4.1 决策追溯

#### 4.1.1 状态快照

HDS 提供完整的状态快照：

| 字段 | 类型 | 说明 |
| :--- | :--- | :--- |
| `hSocial` | Number | 社交饱腹感 |
| `hEnergy` | Number | 认知能量 |
| `da` | Number | 多巴胺水平 |
| `ne` | Number | 去甲肾上腺素水平 |
| `ht` | Number | 血清素水平 |

#### 4.1.2 决策理由

HDS 提供决策理由，使用伪变量表示阈值：

| 理由 | 触发条件 | 影响 |
| :--- | :--- | :--- |
| low_energy | \(H_{\mathrm{energy}} < \tau_{\mathrm{energy,low}}\) | 降低耐心度、话量 |
| high_stress | \(N_{\mathrm{NE}} > \tau_{\mathrm{ne,high}}\) | 提升防御性、直率 |
| low_motivation | \(N_{\mathrm{DA}} < \tau_{\mathrm{da,low}}\) | 降低好奇心、主动性 |
| high_social | \(H_{\mathrm{social}} > \tau_{\mathrm{social,high}}\) | 降低主动性、温暖度 |

**参数解释**：
- \(\tau_{\mathrm{energy,low}}\)：能量低阈值（不展示真实数值）
- \(\tau_{\mathrm{ne,high}}\)：NE 高阈值（不展示真实数值）
- \(\tau_{\mathrm{da,low}}\)：DA 低阈值（不展示真实数值）
- \(\tau_{\mathrm{social,high}}\)：社交饱腹感高阈值（不展示真实数值）

### 4.2 人格演化监控

#### 4.2.1 状态演化轨迹

HDS 记录状态演化轨迹，使用伪变量表示示例值：

| 时间戳 | $H_{social}$ | $H_{energy}$ | \(N_{\mathrm{DA}}\) | \(N_{\mathrm{NE}}\) | \(N_{\mathrm{5HT}}\) |
| :--- | :--- | :--- | :--- | :--- | :--- |
| \(T_0\) | \(\tilde{H}_{\mathrm{social},0}\) | \(\tilde{H}_{\mathrm{energy},0}\) | \(\tilde{N}_{\mathrm{DA},0}\) | \(\tilde{N}_{\mathrm{NE},0}\) | \(\tilde{N}_{\mathrm{5HT},0}\) |
| \(T_1\) | \(\tilde{H}_{\mathrm{social},1}\) | \(\tilde{H}_{\mathrm{energy},1}\) | \(\tilde{N}_{\mathrm{DA},1}\) | \(\tilde{N}_{\mathrm{NE},1}\) | \(\tilde{N}_{\mathrm{5HT},1}\) |
| \(T_2\) | \(\tilde{H}_{\mathrm{social},2}\) | \(\tilde{H}_{\mathrm{energy},2}\) | \(\tilde{N}_{\mathrm{DA},2}\) | \(\tilde{N}_{\mathrm{NE},2}\) | \(\tilde{N}_{\mathrm{5HT},2}\) |

**参数解释**：
- \(\tilde{\cdot}\)：示例占位值（仅用于展示格式，不来自代码/配置）
- \(\tilde{H}_{\mathrm{social},k}\)、\(\tilde{H}_{\mathrm{energy},k}\)：稳态变量示例快照
- \(\tilde{N}_{\mathrm{DA},k}\)、\(\tilde{N}_{\mathrm{NE},k}\)、\(\tilde{N}_{\mathrm{5HT},k}\)：神经调制变量示例快照

#### 4.2.2 慢变量演化

HDS 记录慢变量演化：

| 时间戳 | $\mu_{DA}$ | $\mu_{NE}$ | $\mu_{5HT}$ |
| :--- | :--- | :--- | :--- |
| \(T_0\) | \(\tilde{\mu}_{\mathrm{DA},0}\) | \(\tilde{\mu}_{\mathrm{NE},0}\) | \(\tilde{\mu}_{\mathrm{5HT},0}\) |
| \(T_1\) | \(\tilde{\mu}_{\mathrm{DA},1}\) | \(\tilde{\mu}_{\mathrm{NE},1}\) | \(\tilde{\mu}_{\mathrm{5HT},1}\) |
| \(T_2\) | \(\tilde{\mu}_{\mathrm{DA},2}\) | \(\tilde{\mu}_{\mathrm{NE},2}\) | \(\tilde{\mu}_{\mathrm{5HT},2}\) |
| \(T_3\) | \(\tilde{\mu}_{\mathrm{DA},3}\) | \(\tilde{\mu}_{\mathrm{NE},3}\) | \(\tilde{\mu}_{\mathrm{5HT},3}\) |

**参数解释**：
- \(\tilde{\mu}_{i,k}\)：通道 \(i\) 在时刻 \(T_k\) 的基线示例快照（仅占位，不来自代码/配置）

### 4.3 异常诊断

#### 4.3.1 状态饱和率

状态饱和率监控：

$$
\text{saturationRate} = \frac{N_{saturated}}{N_{total}}
$$

其中：
- \(N_{\mathrm{saturated}}\)：饱和状态数量（变量 \(\ge \tau_{\mathrm{sat,high}}\) 或 \(\le \tau_{\mathrm{sat,low}}\) 的计数）
- $N_{total}$：总状态数量

**诊断**：
- 饱和率 \(> \tau_{\mathrm{sat,warning}}\)：建议检查限幅与映射增益
- 饱和率 \(> \tau_{\mathrm{sat,critical}}\)：建议触发保护策略（冻结慢变量/回滚配置等）

**参数解释**：
- \(\tau_{\mathrm{sat,high}}, \tau_{\mathrm{sat,low}}\)：变量靠近边界的判定阈值（不展示真实数值）
- \(\tau_{\mathrm{sat,warning}}, \tau_{\mathrm{sat,critical}}\)：饱和率告警阈值（不展示真实数值）

#### 4.3.2 恢复半衰期

恢复半衰期监控：

$$
t_{1/2} = \frac{\ln(2)}{\theta}
$$

其中：
- $\theta$：回归速率

**诊断**：
- 半衰期太短：状态变化太快，缺乏惯性
- 半衰期太长：状态变化太慢，缺乏响应性
