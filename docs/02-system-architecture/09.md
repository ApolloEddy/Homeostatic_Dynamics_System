# 02.9 实现备注
### 9.1 Tick 策略

- 按回合 tick + 空闲时补 tick
- 避免后台常驻耗电/耗资源

### 9.2 状态存储

- 本地 SQLite + 可选加密
- 使用 SharedPreferences 实现持久化

### 9.3 Few-shot 策略

- 优先"风格卡片摘要"
- 原文只在预算允许时注入

### 9.4 映射保护

- 所有映射都应 `clamp`
- 记录 `reasons` 供调试
