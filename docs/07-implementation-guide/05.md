# 07.5 性能优化
### 5.1 计算优化

#### 5.1.1 避免重复计算

避免重复计算的策略：

- 使用缓存存储最近计算结果
- 设置缓存过期时间 \(\Delta t_{\mathrm{cache,expiry}}\)（单位：秒；符号化）
- 在缓存有效期内直接返回缓存结果

**参数解释**：
- \(\Delta t_{\mathrm{cache,expiry}}\)：缓存过期时间（不展示真实数值）

#### 5.1.2 批量操作

批量操作的策略：

- 将多个事件合并为一次批量处理
- 减少状态更新的次数
- 提高计算效率

### 5.2 存储优化

#### 5.2.1 延迟保存

延迟保存的策略：

- 标记状态为"脏"（dirty）
- 设置延迟保存定时器 \(\Delta t_{\mathrm{save,delay}}\)（单位：秒；符号化）
- 定时器触发时才执行保存操作

**参数解释**：
- \(\Delta t_{\mathrm{save,delay}}\)：延迟保存时间（不展示真实数值）

#### 5.2.2 压缩存储

压缩存储的策略：

- 使用 gzip 压缩 JSON 数据
- 使用 base64 编码压缩后的数据
- 减少存储空间占用
