# 05.1 概述
### 1.1 定位

L3 长时序人格固化的目标是：减少灾难性遗忘与人格漂移，同时避免"一次极端事件永久绑架人格"。

### 1.2 核心职责

L3 长时序人格固化的核心职责包括：

- 记录交互日志（State-Aware Logging）
- 分箱记忆存储（Binned Memory）
- 离线巩固处理（Sleep Consolidation）
- 慢变量更新（带限幅 + 可复原）

### 1.3 触发条件

L3 的触发条件：

- 闲置/夜间/对话间隔达到阈值
- 手动触发
