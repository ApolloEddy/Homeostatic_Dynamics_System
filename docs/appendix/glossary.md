# 附录 · 术语表

**[返回目录](../INDEX.md)** · **[写作规范](../STYLE.md)** · **[返回主页](../../README.md)**

> [!NOTE]
> 本术语表用于统一白皮书中的术语口径，帮助快速查阅与跨章节引用。

---

| 术语 | 英文 | 解释 |
| :--- | :--- | :--- |
| 内驱力 / 稳态变量 | Homeostasis | 用于描述系统“长期资源/需求”状态的有界变量集合 |
| 异稳态 | Allostasis | 环境变化时，通过改变策略与有效设定点实现长期适应的调节方式 |
| 神经调制变量 | Neuromodulator | 用于描述全局增益/偏置的有界变量集合（如 DA/NE/5HT 通道） |
| 行为向量 | Behavior Vector | 状态到策略之间的可解释中间层连续表征 \(\mathbf{b}\) |
| 控制包 | Control Bundle | 可审计的控制信号集合：注入文本、采样控制、检索偏置与状态快照 |
| OU 过程 | Ornstein–Uhlenbeck Process | 具有均值回归特性的随机过程，常用于表示“惯性 + 波动” |
| 奖赏预测误差 | Reward Prediction Error (RPE) | 用作显著性/学习信号的代理量（工程化定义） |
| 分箱记忆 | Binned Memory | 按类别隔离存储的记忆结构（Fact/Relation/Affect/Style 等） |
| 记忆巩固 | Memory Consolidation | 离线更新/压缩/迁移记忆以提升长期稳定性的过程 |
